title: 身份组
description: 新成员自动获得身份组，或是发一条消息让成员自己点按钮领取身份组。
---

两个命令搞定身份组管理，都需要 **管理身份组** 权限：

| 命令 | 用途 |
| ---- | ---- |
| <Cmd name="auto-role" /> | 新成员加入时自动给身份组 |
| <Cmd name="role-menu" /> | 发一条消息让成员自己领身份组 |

<Warning>
  要发的身份组必须排在机器龙最高身份组的下面，不然机器龙没有权限给予。
</Warning>

<RoleOrderDemo />

不知道怎么调整顺序？参考 [身份组顺序的调整教程](/zh-cn/docs/basics/#身份组的顺序很重要)。

## 新成员自动身份组

<SlashCommand name="auto-role" description="设置新成员自动获得的身份组" />

执行后会弹出表单，选 1 到 2 个身份组，之后每个新加入的成员都会自动获得。

## 自助领取菜单

<SlashCommand name="role-menu" description="发送一条自助领取身份组的消息" />

执行后会弹出表单：

| 字段 | 说明 |
| ---- | ---- |
| 样式 | 按钮或下拉菜单 |
| 消息内容 | 最多 4000 字，支持 Markdown |
| 身份组 | 最多 25 个 |
