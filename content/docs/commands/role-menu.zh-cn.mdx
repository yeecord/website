<Cmd name="role-menu" /> 发一条带按钮或下拉菜单的消息，成员自己点就能领身份组。

<CommandHeader you={["管理身份组"]} bot={["管理身份组"]} />

## 创建菜单

<SlashCommand name="role-menu" description="发送一条自助领取身份组的消息" />

执行后会弹出表单：

| 字段 | 说明 |
| ---- | ---- |
| 样式 | 按钮或下拉菜单 |
| 消息内容 | 最多 4000 字，支持 Markdown |
| 身份组 | 最多 25 个 |

<DiscordChat>
  <DiscordMessage>
    <p>点下面的按钮领取你的身份组！</p>
    <p>
      <DiscordButton>🔔 活动通知</DiscordButton>{" "}
      <DiscordButton>🎮 游戏玩家</DiscordButton>{" "}
      <DiscordButton>🎨 创作者</DiscordButton>
    </p>
  </DiscordMessage>
</DiscordChat>

成员点按钮获得身份组，再点一次取消。

<Warning>
  要发的身份组必须排在机器龙最高身份组的下面，不然机器龙没有权限给予，
  详见 [身份组顺序的调整教程](/zh-cn/docs/basics/#身份组的顺序很重要)。
</Warning>

## 想改成自动发放？

新成员一进来就该有的身份组，用 [新成员身份组](/zh-cn/docs/commands/auto-role/) 自动发。
