<Cmd name="poll" /> 是機器龍最強的功能之一。投票結果會即時畫成一張圖片，每次有人投票都會更新，還支援匿名、鎖票、開票前保密這些進階選項。

把機器龍 [安裝到帳號](/docs/#兩種安裝方式) 之後，投票在私訊、群組私訊、任何伺服器都能發起。

## 發起投票

<SlashCommand name="poll create" description="發起投票，會跳出表單讓你填內容" />

送出指令後會跳出表單：

| 欄位 | 說明 |
| ---- | ---- |
| 標題 | 最多 100 字 |
| 選項 | 一行一個，2 到 18 個選項 |
| 描述 | 選填，最多 1000 字 |
| 結束時間 | 選填，例如 `30m`、`2h`、`3d`，留空表示手動結束 |

表單裡還有一個進階選項選單，可以複選：

| 選項 | 效果 |
| ---- | ---- |
| 匿名投票 | 結果只顯示票數，看不到誰投了什麼 |
| 投票後不能改票 | 投下去就不能反悔 |
| 開票前保密 | 結束前所有人（包含你）都看不到票數 |

## 管理投票

每個子指令的 `poll` 參數都有自動補全，輸入時直接選你的投票就好。

| 指令 | 說明 |
| ---- | ---- |
| <Cmd name="poll edit" /> | 修改標題、描述、結束時間、圖片，或補開匿名等設定 |
| <Cmd name="poll end" /> | 結束投票 |
| <Cmd name="poll resend" /> | 重新傳送投票訊息（原本的訊息被洗版時很好用） |
| <Cmd name="poll remove-vote" /> | 移除某位成員的票 |
| <Cmd name="poll role-weight" /> | 設定身分組的票數權重 |
| <Cmd name="poll export" /> | 匯出結果 |

## 身分組權重

<SlashCommand
  name="poll role-weight"
  options={[
    { name: "poll", value: "poll", required: true },
    { name: "role", value: "@幹部", required: true },
    { name: "weight", value: "2", required: true },
  ]}
  description="讓某個身分組的成員一票抵多票"
/>

權重範圍 0 到 100。設 0 可以讓某個身分組的票不算數，設 2 就是一票抵兩票。

## 匯出結果

<Cmd name="poll export" /> 會同時給你 CSV 跟 Excel 兩種檔案，欄位包含選項、投票者、投票者 ID、權重、投票時間。

投票途中或結束後都能匯出。

<Info>
  CSV 直接用 Excel 開啟會讓投票者 ID 溢位變成科學記號，建議直接用附上的
  `.xlsx` 檔。
</Info>

## 舊版投票

用舊版指令發起的投票按鈕仍然可以正常運作，不用擔心升級後壞掉。
