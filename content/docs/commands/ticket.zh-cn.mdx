---
title: 私人频道
description: 让成员一键开启只有自己和管理员看得到的私人讨论串，反馈问题不用私信管理员。
---

想让成员反馈问题、联系管理员，又不想开放私信？<Cmd name="ticket" /> 会发一条带按钮的消息，成员点按钮就开一个只有他和管理员看得到的私人讨论串。需要 **管理频道** 权限。

## 设置

<SlashCommand name="ticket setup" description="初始化私人频道功能" />

执行后会弹出表单：

| 字段 | 说明 |
| ---- | ---- |
| 消息内容 | 按钮消息的文字，最多 4000 字 |
| 通知身份组 | 开新讨论串时要提及的身份组，1 到 5 个 |
| 发送频道 | 按钮消息要放在哪个频道 |

<DiscordChat>
  <DiscordMessage>
    <p>需要帮助吗？点下面的按钮开启私人频道，管理员会尽快回复你。</p>
    <p>
      <DiscordButton variant="primary">🎫 开启私人频道</DiscordButton>
    </p>
  </DiscordMessage>
</DiscordChat>

成员点按钮后会开一个私人讨论串，并提及你设置的管理身份组。

## 卸载

<Cmd name="ticket uninstall" /> 移除设置，已开的讨论串不受影响。
