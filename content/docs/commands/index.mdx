在輸入框打 `/` 就會列出機器龍的指令，Discord 會提示每個參數的名稱跟說明，必填參數不填會直接擋下來。

<SlashCommand
  name="poll create"
  description="輸入 / 之後選擇指令，Discord 會引導你填完參數"
/>

以前的 `&` 文字指令已經退役。輸入舊指令時，機器龍會告訴你對應的斜線指令是哪一個。

## 在哪裡可以用

把機器龍 [安裝到帳號](../index.mdx#兩種安裝方式) 之後，標了「到處可用」的指令在私訊、群組私訊、任何伺服器都能使用；其他指令只能在裝了機器龍的伺服器裡用。

## 伺服器管理

| 指令 | 說明 | 需要權限 |
| ---- | ---- | -------- |
| [<Cmd name="auto-channels" />](./auto-channels.mdx) | 動態語音頻道 | 管理頻道 |
| [<Cmd name="lock-channel" />](./lock-channel.mdx) | 密碼鎖定語音頻道 | 管理頻道 |
| <Cmd name="afk-channel" /> | 設定 AFK 語音頻道 | 管理伺服器 |
| [<Cmd name="clear" />](./clear.mdx) | 大量刪除訊息 | 管理訊息 |
| [<Cmd name="bot-fight" />](./bot-fight.mdx) | 機器人防禦模式 | 封鎖成員 |
| [<Cmd name="auto-role" />](./auto-role.mdx) | 新成員自動身分組 | 管理身分組 |
| [<Cmd name="role-menu" />](./role-menu.mdx) | 自助領取身分組選單 | 管理身分組 |
| [<Cmd name="member-notification" />](./member-notification.mdx) | 成員加入／離開通知 | 管理伺服器 |
| [<Cmd name="form" />](./form.mdx) | 表單與回報系統 | 管理頻道 |
