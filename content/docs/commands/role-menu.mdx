<Cmd name="role-menu" /> 發一則帶按鈕或下拉選單的訊息，成員自己點就能領身分組。

<CommandHeader you={["管理身分組"]} bot={["管理身分組"]} />

## 建立選單

<SlashCommand name="role-menu" description="傳送一則自助領取身分組的訊息" />

執行後會跳出表單：

| 欄位 | 說明 |
| ---- | ---- |
| 樣式 | 按鈕或下拉選單 |
| 訊息內容 | 最多 4000 字，支援 Markdown |
| 身分組 | 最多 25 個 |

<RoleMenuDemo />

成員點按鈕獲得身分組，再點一次取消。

<Warning>
  要發的身分組必須排在機器龍最高身分組的下面，不然機器龍沒有權限給予，
  詳見 [身分組順序的調整教學](../basics.mdx#身分組的順序很重要)。
</Warning>

## 想改成自動發放？

新成員一進來就該有的身分組，用 [新成員身分組](./auto-role.mdx) 自動發。
