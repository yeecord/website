在输入框打 `/` 就会列出机器龙的命令，Discord 会提示每个参数的名称和说明，必填参数不填会直接拦下来。

  name="poll create"
  description="输入 / 之后选择命令，Discord 会引导你填完参数"
/>

以前的 `&` 文字命令已经退役。输入旧命令时，机器龙会告诉你对应的斜杠命令是哪一个。

## 在哪里可以用

把机器龙 [安装到账号](/zh-cn/docs/#两种安装方式) 之后，标了「到处可用」的命令在私信、群组私信、任何服务器都能使用；其他命令只能在装了机器龙的服务器里用。

## 服务器管理

| 命令 | 说明 | 需要权限 |
| ---- | ---- | -------- |
| [<Cmd name="auto-channels" />](/zh-cn/docs/commands/auto-channels/) | 动态语音频道 | 管理频道 |
| [<Cmd name="lock-channel" />](/zh-cn/docs/commands/lock-channel/) | 密码锁定语音频道 | 管理频道 |
| <Cmd name="afk-channel" /> | 设置 AFK 语音频道 | 管理服务器 |
| [<Cmd name="clear" />](/zh-cn/docs/commands/clear/) | 大量删除消息 | 管理消息 |
| [<Cmd name="bot-fight" />](/zh-cn/docs/commands/bot-fight/) | 机器人防御模式 | 封禁成员 |
| [<Cmd name="auto-role" />](/zh-cn/docs/commands/role/) | 新成员自动身份组 | 管理身份组 |
| [<Cmd name="role-menu" />](/zh-cn/docs/commands/role/) | 自助领取身份组菜单 | 管理身份组 |
| [<Cmd name="member-notification" />](/zh-cn/docs/commands/member-notification/) | 成员加入／退出通知 | 管理服务器 |
| [<Cmd name="ticket" />](/zh-cn/docs/commands/ticket/) | 私人频道 | 管理频道 |
