---
title: 指令清單
---

透過使用指令，你能夠跟機器人進行互動，並且同時又不干擾到聊天，使用 `&` 開頭呼叫這些指令，或是標記 `@YEE式機器龍` 也可以。

因為文檔還在努力進行更新，要查詢最新的指令訊息可以使用 `&help` 指令，並點選下方 `指令教學` 的下拉式選單來查閱。

![幫助指令](/docs/commands/help.png)

## 一般

| 指令                             | 描述                                    |
| -------------------------------- | --------------------------------------- |
| &avatar [使用者標記]             | 取得使用者頭像                          |
| &banner [使用者標記]             | 取得使用者旗幟                          |
| &botinfo                         | 查詢機器人資料                          |
| &bullshit \<主題> [字數]         | 唬爛產生器                              |
| &clear \<數量>                   | 清除訊息(最多 1000)                     |
| &credit                          | 查看各位心臓捧げよ的工作人員            |
| &feedback                        | 填寫表單給我們回饋                      |
| &guildinfo \<伺服器 ID>          | 查詢伺服器資                            |
| &help [指令名稱]                 | 查詢指令使用方法                        |
| &minecraft [伺服器 IP/玩家 ID]   | 查詢 Minecraft 伺服器/玩家/Hypixel 資訊 |
| &pick \<問題名稱>                | 幫你選個答案                            |
| &ping                            | 戳我然後看看我是不是活著                |
| &time \<時間格式>                | 查詢時間轉換成時間戳記                  |
| &userinfo [使用者標記/使用者 ID] | 查詢使用者資訊                          |
