---
title: 私人頻道
description: 讓成員一鍵開啟只有自己和管理員看得到的私人討論串，回報問題不用私訊管理員。
---

想讓成員回報問題、聯絡管理員，又不想開放私訊？<Cmd name="ticket" /> 會發一則帶按鈕的訊息，成員點按鈕就開一個只有他和管理員看得到的私人討論串。需要 **管理頻道** 權限。

## 設定

<SlashCommand name="ticket setup" description="初始化私人頻道功能" />

執行後會跳出表單：

| 欄位 | 說明 |
| ---- | ---- |
| 訊息內容 | 按鈕訊息的文字，最多 4000 字 |
| 通知身分組 | 開新討論串時要標註的身分組，1 到 5 個 |
| 發送頻道 | 按鈕訊息要放在哪個頻道 |

<DiscordChat>
  <DiscordMessage>
    <p>需要協助嗎？點下面的按鈕開啟私人頻道，管理員會盡快回覆你。</p>
    <p>
      <DiscordButton variant="primary">🎫 開啟私人頻道</DiscordButton>
    </p>
  </DiscordMessage>
</DiscordChat>

成員點按鈕後會開一個私人討論串，並標註你設定的管理身分組。

## 解除安裝

<Cmd name="ticket uninstall" /> 移除設定，已開的討論串不受影響。
