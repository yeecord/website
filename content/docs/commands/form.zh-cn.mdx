<Cmd name="form" /> 让你设计自己的表单。成员在面板上按一颗按钮、填完提交，机器龙就开一个私密子区，把填表的人和管理员一起拉进去，后续直接在里面谈。

举报表单、进服申请、客服单，都是同一套流程。整个过程是这样：

## 创建表单

<SlashCommand name="form create" description="弹出表单，填标题和通知对象" />

发送命令后填这几个字段：

| 字段 | 说明 |
| ---- | ---- |
| 标题 | 最多 45 字，也是成员填表窗口的标题 |
| 说明 | 选填，会显示在面板消息上 |
| 按钮文字 | 选填，不填就叫「填写」 |
| 通知身份组 | 1 到 5 个，有人提交表单时会提及他们 |

建好之后会出现一张只有你看得到的管理卡。加题目、发布面板、停用，全部在这张卡上操作，卡片会原地更新，不会刷屏：

## 加题目

在管理卡上按「新增问题」，一次加一题，最多 5 题（Discord 表单窗口的字段上限）。

题目分三种：

| 类型 | 说明 |
| ---- | ---- |
| 短答 | 一行文字 |
| 段落 | 多行文字，适合「请描述发生什么事」这种题目 |
| 单选 | 下拉菜单，选项填 2 到 25 个，用逗号或换行分隔 |

发布之后照样能加题、删题，不用重发面板。成员每次点按钮，机器龙都会拿最新的题目组表单。

## 发布面板

在管理卡上按「发布面板」，选一个频道，机器龙就把面板发过去。

## 成员提交之后

面板频道底下会多一个私密子区，名字像 `#0042 阿猫`，编号按提交顺序排，扫一眼子区列表就知道有几张单。

机器龙会在子区里提及填表的人和通知身份组，答案整理成一张卡。频道里的其他人看不到这个子区。

谈完之后按「结案」，子区会归档并锁定。只有通知身份组的成员、或有「管理子区」权限的人按得动。

## 管理表单

<SlashCommand name="form list" description="列出这个服务器的表单，选一张打开管理卡" />

每个服务器最多 10 张表单。停用之后面板按钮还在，只是成员点了会被告知表单已停用；再按一次「启用」就恢复。
