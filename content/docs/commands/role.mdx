---

兩個指令搞定身分組管理，都需要 **管理身分組** 權限：

| 指令 | 用途 |
| ---- | ---- |
| <Cmd name="auto-role" /> | 新成員加入時自動給身分組 |
| <Cmd name="role-menu" /> | 發一則訊息讓成員自己領身分組 |

<Warning>
  要發的身分組必須排在機器龍最高身分組的下面，不然機器龍沒有權限給予。
</Warning>

<RoleOrderDemo />

不知道怎麼調整順序？參考 [身分組順序的調整教學](/docs/basics/#身分組的順序很重要)。

## 新成員自動身分組

<SlashCommand name="auto-role" description="設定新成員自動獲得的身分組" />

執行後會跳出表單，選 1 到 2 個身分組，之後每個新加入的成員都會自動獲得。

## 自助領取選單

<SlashCommand name="role-menu" description="傳送一則自助領取身分組的訊息" />

執行後會跳出表單：

| 欄位 | 說明 |
| ---- | ---- |
| 樣式 | 按鈕或下拉選單 |
| 訊息內容 | 最多 4000 字，支援 Markdown |
| 身分組 | 最多 25 個 |

<DiscordChat>
