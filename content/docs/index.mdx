YEE 式機器龍是一隻全中文的 Discord 機器人。所有功能都用斜線指令操作，介面、說明、錯誤訊息都是中文。

## 兩種安裝方式

| 安裝方式 | 說明 |
| -------- | ---- |
| 加入伺服器 | 整個伺服器的成員都能使用 |
| 安裝到帳號 | 裝在你自己的 Discord 帳號上，私訊、群組私訊、任何伺服器都能用 |

點開邀請連結後 Discord 會讓你選，兩種可以同時裝。裝到帳號之後，<Cmd name="poll" />、<Cmd name="profile" />、<Cmd name="find-food" /> 這類指令在沒有機器龍的地方也能使用。
