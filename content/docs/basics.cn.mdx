## 几个名词

| 名词 | 意思 |
| ---- | ---- |
| 服务器 | 一个聊天社群，像一个微信群再大很多，里面可以开很多频道 |
| 频道 | 服务器里的一个聊天区，`#` 开头是文字频道，🔊 开头是语音频道 |
| 身份组 | 贴在成员身上的标签，可以决定名字颜色和能做什么事 |
| 权限 | 「谁能做什么」的开关，可以设在身份组上，也可以设在单个频道上 |
| 斜杠命令 | 在输入框打 `/` 呼叫机器人的功能，Discord 会一步步引导你 |

## 身份组的顺序很重要

服务器设置里的身份组是一个列表，**上面的管下面的**。机器龙要给成员发身份组时，只能发排在它下面的。所以文档里常常提醒你：把机器龙的身份组拖到上面。

打开方式：点服务器名称 → **服务器设置** → **身份组**，按住拖拽就能调整顺序。

每个频道可以单独设置「谁看得到、谁能发言」。文档里说「把频道设成只读」，就是下面这个操作：

打开方式：鼠标移到频道名称上 → 点齿轮 ⚙️ → **权限**。

## 机器龙说我权限不足？

两种可能：

- **你**没有权限：命令需要的权限（例如「管理频道」）你身上没有，找服务器管理员
- **机器龙**没有权限：去服务器设置 → 身份组，点「YEE式機器龍」，把该开的权限打开

每个功能页面的开头都写了它需要哪些权限。
