## 幾個名詞

| 名詞 | 意思 |
| ---- | ---- |
| 伺服器 | 一個聊天社群，像一個 LINE 群組再大很多，裡面可以開很多頻道 |
| 頻道 | 伺服器裡的一個聊天區，`#` 開頭是文字頻道，🔊 開頭是語音頻道 |
| 身分組 | 貼在成員身上的標籤，可以決定名字顏色和能做什麼事 |
| 權限 | 「誰能做什麼」的開關，可以設在身分組上，也可以設在單一頻道上 |
| 斜線指令 | 在輸入框打 `/` 呼叫機器人的功能，Discord 會一步步引導你 |

## 身分組的順序很重要

伺服器設定裡的身分組是一個列表，**上面的管下面的**。機器龍要發身分組給成員時，只能發排在它下面的。所以文件裡常常提醒你：把機器龍的身分組拖到上面。

<RoleOrderDemo />

打開方式：點伺服器名稱 → **伺服器設定** → **身分組**，按住拖曳就能調整順序。

## 頻道權限怎麼設

每個頻道可以單獨設定「誰看得到、誰能發言」。文件裡說「把頻道設成唯讀」，就是下面這個操作：

<ChannelPermissionDemo />

打開方式：滑鼠移到頻道名稱上 → 點齒輪 ⚙️ → **權限**。

## 機器龍的權限怎麼改

機器龍的權限在 [安裝](https://yeecord.com/install) 的時候決定：安裝頁勾了哪些功能，就會要求對應的權限。事後要調整有兩個方法：

- 伺服器設定 → 身分組 → 點「YEE式機器龍」，直接開關權限
- 重新點一次安裝連結，改勾需要的功能再授權一次（不用先踢出機器龍）

每個功能頁面的開頭都有一張卡片，寫著「誰能用」跟「機器龍需要」哪些權限。

## 機器龍說我權限不足？

兩種可能：

- **你**沒有權限：指令需要的權限（例如「管理頻道」）你身上沒有，找伺服器管理員
- **機器龍**沒有權限：照上面的方法把該開的權限打開
