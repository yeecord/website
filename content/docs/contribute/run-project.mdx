description: 在自己的電腦上運行文件網站，即時預覽你的修改。
---

網站用 [fumapress](https://press.fumadocs.dev) + [Waku](https://waku.gg) 打造，內容都是 MDX 檔案。改文件其實不需要跑起來，但想即時預覽的話照下面做。

## 需要的工具

- 任何程式編輯器，推薦 [Visual Studio Code](https://code.visualstudio.com/)
- [Bun](https://bun.sh)：這個專案的執行環境跟套件管理器

安裝 Bun 之後確認一下版本：

```bash
bun -v
```

## 跑起來

```bash
git clone https://github.com/yeecord/website.git
cd website
bun install
bun dev
```

打開終端機顯示的網址就能看到網站，改動 `content/` 裡的檔案會即時重新整理。

## 建置檢查

送 PR 前跑一次，確定沒有壞掉的型別或連結：

```bash
bun run typecheck
bun run build
```
