## 覓食

選一個地點帶 Yee 出門。袋子會隨時間慢慢裝滿，裝滿大約要一天半到兩天，收成時揭曉這趟撿到什麼。

- 食材有普通、稀有、傳說三種稀有度
- 每個地點每天輪換情報，提示今天容易撿到什麼
- 有保底機制：連續幾次沒遇到新食材，之後幾次內必定遇到
- 偶爾會撿到驚喜，也會看到其他玩家最近的收穫足跡
- 路上會碰到小事件，選不同的分支有不同結果

## 烹飪

從背包挑幾樣食材下鍋，Yee 會評分：

| 評級 | 說明 |
| ---- | ---- |
| 普通 | 能吃 |
| 好吃 | 食材搭配得不錯 |
| 絕品 | 完美組合 |

搭配得太離譜會煮出 **黑暗料理** — 但別急著難過，黑暗料理有自己的隱藏菜譜可以收集。

每天有一個「今日想吃」的風味提示，只給方向不給答案，照著方向煮容易拿高分。

## 菜譜與餵養

- **菜譜** 記錄你煮出過的所有料理，收集控的天堂
- **餵牠** 把料理餵給 Yee 培養感情，感情越好互動越多
- **分享成績** 產生一張成績卡片跟朋友炫耀

## 銅板

覓食和烹飪會賺到銅板，是找吃的世界裡的貨幣。跟舊 RPG 不同，銅板只在遊戲內流通，沒有玩家間交易。
