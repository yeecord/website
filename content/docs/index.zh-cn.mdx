YEE 式机器龙是一只全中文的 Discord 机器人。所有功能都用斜杠命令操作，界面、说明、错误消息都是中文。

## 两种安装方式

| 安装方式 | 说明 |
| -------- | ---- |
| 加入服务器 | 整个服务器的成员都能使用 |
| 安装到账号 | 装在你自己的 Discord 账号上，私信、群组私信、任何服务器都能用 |

点开邀请链接后 Discord 会让你选，两种可以同时装。装到账号之后，<Cmd name="poll" />、<Cmd name="profile" />、<Cmd name="find-food" /> 这类命令在没有机器龙的地方也能使用。
